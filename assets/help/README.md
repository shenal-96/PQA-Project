# Help assets

PNG screenshots shown inside the in-app Help dialog. Drop the following
files into this directory (exact filenames) for the images to appear:

- `step2_select_ll_voltage.png` — PQone Option dialog with "Line – Line"
  selected for Urms.
- `step3_trend_graph_export.png` — PQone CSV export dropdown with
  "Trend Graph" highlighted.
- `step4_export_dialog.png` — "Export csv file – Trend Graph" dialog with
  U/I, Frequency, Harmonics – Trend, Power, and Energy ticked.

If a file is missing the dialog renders a placeholder in its place; no
code change required when assets are added.
